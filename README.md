# pyFuel
Si connette al sito del MIM e scarica i prezzi dei distributori di carburante.

Poi vi chiede di inserire il vostro indirizzo e la distanza del raggio di ricerca

Nel browser appariranno tutti i distributori, con evidenziati in verde quelli con il prezzo inferiore per benzina e gasolio

## How to use
Utilizza il virtual environment di Python3
Andare nella cartella scaricata e digitare:

> source bin/activate

> bash pyFuel.sh
